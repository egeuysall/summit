# Summit

**Summit** is a **gamified task marketplace** where colonists complete missions to earn **credits**.  
Accumulate enough credits to unlock **interplanetary travel rewards**, turning everyday skill-sharing into an adventure across the solar system.

> “Work hard on the Moon, explore the solar system.”

## What It Is

Summit connects colonists who need help with those eager to contribute.  
Complete tasks, earn credits, and climb the leaderboard to unlock cosmic destinations like **Mars**, **Europa**, and **Titan**.

## Goal
Create a **motivating, collaborative, and fun** 24-hour hackathon prototype that blends **productivity, play, and purpose**.

## MVP Features (24-Hour Scope)

### User Profiles
- Display name, avatar, skill tags (from preset list), and credit balance  

### Task Posting
- Post help requests with title, description, required skill, urgency badge, and credit reward  

### Browse Requests Feed
- Filter tasks by skill  
- Sort by credit reward  

### Task Claim & Completion
- “I can help” button to claim  
- Completion confirmed by requester → credits transferred  

### Credit System
- Earn credits for completed tasks  
- View top earners in leaderboard  

### Reward Shop
- Unlock destinations with credits (Mars, Europa, Titan)
